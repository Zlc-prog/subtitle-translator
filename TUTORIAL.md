# SubEditor 使用教程

SubEditor 是一款 macOS 桌面字幕处理工具，基于 AI 提供字幕翻译、字幕分割、字幕修改、社交媒体贴文生成等功能。

## 1. 安装与 API Key 设置

### 安装

从 [GitHub Releases] 下载最新版本的 `.dmg` 文件，双击安装。

### 设置 API Key

1. 点击左侧边栏底部的 **API Key**
2. 输入你的 DeepSeek API Key（格式为 `sk-...`）
3. 点击 **保存**

> API Key 仅存储在本地，不会上传至任何服务器。重启应用后无需重新输入。

## 2. 字幕修改

字幕修改是 SubEditor 的核心功能，用于查看和编辑 SRT 字幕文件。

### 基本操作

- **上传 SRT**：点击「上传 SRT」按钮，或直接将 `.srt` 文件拖拽到窗口
- **编辑文本**：直接点击字幕文本区域进行修改
- **编辑时间码**：点击时间码输入框修改起止时间
- **拆分字幕**：在文本光标位置按 `Enter`，将一行拆分为两行
- **合并字幕**：在句首位置按 `Backspace`，与上一行合并
- **删除字幕**：鼠标悬停在行号上方，点击出现的 `✕` 按钮
- **撤销**：点击「↩ 撤回」或按 `Cmd+Z`

### 导出

- **导出 SRT**：保存为标准字幕文件
- **导出 TXT**：仅导出纯文本内容

[📸 截图：字幕修改主界面]

### 参考字幕（双语对照）

参考字幕功能帮助你在修改双语字幕时保持对齐。

#### SRT 参考模式

点击「参考字幕」→「上传 SRT 参考」，选择参考 SRT 文件。界面分为左右两栏：

- **左栏**：参考字幕（只读）
- **右栏**：编辑字幕（可修改）

两侧同步滚动，可逐行对照修改。

[📸 截图：SRT 参考模式双栏布局]

#### 纯文本参考模式

点击「参考字幕」→「上传 TXT 文本」或「粘贴纯文本」，左侧显示完整参考文本，独立滚动，不受编辑列影响。

[📸 截图：文本参考模式]

#### 空白行占位

当编辑列进行拆分/合并操作后，两边行数可能不一致。点击行号旁的 `+` 按钮插入空白占位行，手动对齐。

> 空白行不会出现在导出的字幕文件中。

[📸 截图：空白行占位功能]

## 3. 字幕翻译

使用 AI 将中文字幕翻译为自然、口语化的英文字幕。

### 使用步骤

1. 上传 SRT 字幕文件
2. 确认 API Key 已配置
3. 点击「一键翻译」
4. 等待 AI 逐句翻译完成

### 翻译规则

点击「翻译规则」可自定义：

- **专有名词词典**：中 → 英对照表，确保术语翻译一致性
- **自定义指令**：补充额外的翻译要求

### 审核专有名词

翻译完成后，包含专有名词标记（`[名词：xxx，请人工核校]`）的字幕会高亮显示。点击「审核」按钮逐条确认：

- **接受**：去除标记，保留翻译
- **修改**：手动编辑译文
- **跳过**：保留标记，继续下一条

[📸 截图：翻译主界面]
[📸 截图：审核专有名词弹窗]

### 重新翻译

点击单行右侧的「重新翻译」按钮可对某个句组重新翻译。AI 会使用不同表达方式生成新版本。

### AI 免责声明

导出 SRT/TXT 前会弹出提示："AI 结果仅供参考，请务必手动编辑核验"。确认后才会弹出保存路径。

## 4. 字幕分割

将长篇文本自动分割为字幕格式，并自动计算时间戳。

### 使用步骤

1. 在左侧文本区粘贴或输入文字
2. 配置参数：
   - **每行字数上限**：控制单行字幕长度
   - **时长模式**：语速模式（WPM）或固定时长
3. 点击「生成字幕」
4. 预览分割结果
5. 导出为 SRT 文件

[📸 截图：字幕分割界面]

## 5. 标题和贴文生成

基于字幕内容，AI 自动生成视频标题和社交媒体贴文。

### 配置面板

- **平台**：Instagram / FlyOverChina
- **贴文类型**：默认 / 新闻向 / 故事向 / 营销向
- **语言**：英文 / 中文
- **摄影师署名**：自动添加到贴文中

### 素材来源

- **上传文件**：支持 `.srt` `.txt` `.docx`
- **从软件获取**：直接从字幕翻译/字幕修改/字幕分割的结果中导入
- **手动输入**：直接在文本框粘贴

### 标题生成

点击「生成标题」生成多个备选标题，选择一个后点击「复制选中」。

### 历史记录

标题和贴文均支持历史记录功能。每次重新生成后，旧版本自动保存到历史记录中。点击「历史生成记录」可展开查看和对比。

## 6. 快捷键参考

| 快捷键 | 功能 | 适用页面 |
|--------|------|----------|
| `Cmd+Z` | 撤销 | 全局（非编辑状态） |
| `Cmd+S` | 导出 SRT | 字幕修改 |
| `Enter` | 拆分字幕行 | 字幕修改（光标在文本中） |
| `Backspace` | 合并到上一行 | 字幕修改（光标在行首） |
| `Cmd+Enter` | 保存编辑 | 翻译审核弹窗 |
| `Escape` | 取消编辑 | 翻译审核弹窗 |

---

> AI 生成结果仅供参考，请务必手动编辑核验内容的准确性、合规性和适用性后再对外发布。
