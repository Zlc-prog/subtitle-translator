SubEditor 使用教程

AI 驱动的字幕处理工具 — 翻译、编辑、分割、贴文生成

1. 安装与 API Key 设置

安装

  1. 从 GitHub Releases 下载最新 .dmg 文件
  2. 双击打开 .dmg，将 SubEditor 拖入 Applications 文件夹

如果提示「文件已损坏」或「无法验证开发者」

由于未经过 Apple 官方签名，首次打开可能出现以下提示：

  • "SubEditor" 已损坏，无法打开
  • 无法打开"SubEditor"，因为无法验证开发者

解决方法（二选一）：

方法一：右键打开

  1. 在 Applications 文件夹中找到 SubEditor
  2. 右键点击（或按住 Control 点击）→ 选择「打开」
  3. 弹出提示框后，点击「打开」按钮即可

方法二：终端解除隔离属性

  1. 打开「终端」App（在 Applications / Utilities 中）
  2. 粘贴以下命令并回车：
xattr -cr /Applications/SubEditor.app
  1. 完成后即可正常打开 SubEditor
此命令仅移除 macOS 对未签名应用的安全标记，不会修改应用本身。

设置 API Key

  1. 点击左侧边栏底部 API Key
  2. 输入 DeepSeek API Key（sk-...）
  3. 点击 保存
API Key 仅存储在本地，重启后自动加载，无需重复输入。

2. 字幕修改

核心功能，用于查看和编辑 SRT 字幕文件。

基本操作

  • 上传：点击按钮或拖拽 .srt 文件到窗口
  • 编辑文本：点击字幕文本区域直接修改
  • 编辑时间码：点击时间码修改起止时间
  • 拆分：光标在文本中按 Enter
  • 合并：光标在句首按 Backspace
  • 删除：悬停行号上方，点击 ✕
  • 撤销：点击「↩ 撤回」或 Cmd+Z
  • 导出：SRT / TXT 两种格式

参考字幕（双语对照）

点击「参考字幕」下拉菜单选择模式：

SRT 参考模式

「上传 SRT 参考」→ 左右双栏布局，统一滚动，逐行对照。

  • 左栏：参考字幕（只读）
  • 右栏：编辑字幕（可修改）

纯文本参考模式

「上传 TXT 文本」或「粘贴纯文本」→ 左侧完整文本区，独立滚动。

空白行占位

点击行号旁 + 插入空白行，手动对齐。导出时自动跳过空白行。

3. 字幕翻译

AI 将中文翻译为口语化英文字幕。

流程

  1. 上传 SRT → 点击 一键翻译
  2. 设置 翻译规则（专有名词、自定义指令）
  3. 审核专有名词标记，逐条确认或修改
  4. 导出 SRT / TXT

4. 字幕分割

长文本 → 带时间戳的字幕行。

  1. 粘贴文字 → 设置每行字数上限和计时模式
  2. 点击「生成字幕」→ 预览结果
  3. 导出 SRT

5. 标题和贴文生成

基于字幕内容，AI 生成标题和社交媒体贴文。

  • 配置平台/类型/语言/摄影师署名
  • 素材来源：上传文件 / 从软件获取 / 手动输入
  • 多标题备选，一键复制
  • 历史记录：每次重新生成后旧版自动保存，可展开对比

6. 快捷键

快捷键功能页面
Cmd+Z撤销全局
Cmd+S导出 SRT字幕修改
Enter拆分字幕行字幕修改
Backspace合并到上一行字幕修改
Cmd+Enter保存编辑翻译审核
Escape取消编辑翻译审核

AI 生成结果仅供参考，请务必手动编辑核验
